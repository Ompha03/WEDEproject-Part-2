# WEDEproject-Part-2 For Made in Venda Clothing Brand

This website was made for a clothing brand called Made In Venda. The website aims to make it easier for customers to be able to purchase 
Items from the store without going to the physical store.

It also aims to educate people about the Vhavenda tribe since it is a minority in South Africa, and people do not know a lot about it.

For this part, CSS has been applied to my previous submission
Background colours and images were added, as well as classes to the code to help separate information and images on the home page, and also help with 
styling the inventory on the store page.

In the contact page, a location link was added that takes the customer to Google Maps, where they can see the physical store's location
If they wished to go there.
